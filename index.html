Variables y tipos

Explicación

Las variables sirven para guardar información. En JavaScript podemos usar let, const y var.

Ejemplo

let nombre = "Alex";
let edad = 20;

console.log(nombre);
console.log(edad);

Explicación del ejemplo

Se crean dos variables: una guarda texto y otra un número.

Operadores

Explicación

Los operadores permiten hacer cálculos y comparaciones. 

let a = 5;
let b = 2;

console.log(a + b);
console.log(a > b);

Explicación del ejemplo

El código suma dos números y luego compara cuál es mayor.

Condicionales

Explicación

Los condicionales permiten tomar decisiones dependiendo de una condición. 

let edad = 18;

if (edad >= 18) {
  console.log("Mayor de edad");
} else {
  console.log("Menor de edad");
}

Explicación del ejemplo

Si la edad es mayor o igual a 18 muestra un mensaje diferente.

Bucles

Explicación

Los bucles repiten código varias veces. 

for(let i = 1; i <= 5; i++) {
  console.log(i);
}

Explicación del ejemplo

El bucle muestra los números del 1 al 5.

Arrays

Explicación

Los arrays guardan varios valores dentro de una sola variable. 

let colores = ["rojo", "azul", "verde"];

console.log(colores[0]);

Explicación del ejemplo

Se crea un array y se muestra el primer elemento.

Funciones

Explicación

Las funciones agrupan código para reutilizarlo. 

function saludar(nombre) {
  return "Hola " + nombre;
}

console.log(saludar("Alex"));

Explicación del ejemplo

La función recibe un nombre y devuelve un saludo.

DOM

Explicación

El DOM permite modificar elementos HTML desde JavaScript.

Texto original

function cambiarTexto() {
  document.getElementById("textoDOM").innerHTML = "Texto cambiado";
}

Explicación del ejemplo

Al pulsar el botón cambia el contenido del párrafo.

Eventos

Explicación

Los eventos detectan acciones del usuario como clics o teclas. 

boton.addEventListener("click", function() {
  mensaje.innerHTML = "Has hecho clic";
});

Explicación del ejemplo

Cuando el usuario pulsa el botón aparece un mensaje.

Math

Explicación

Math incluye funciones matemáticas. 

let numero = Math.random();

console.log(numero);

Explicación del ejemplo

Genera un número aleatorio.

Strings

Explicación

Los strings son cadenas de texto. 

let texto = "JavaScript";

console.log(texto.length);

Explicación del ejemplo

Se muestra la longitud del texto.

Fechas

Explicación

JavaScript permite trabajar con fechas y horas. 

let fecha = new Date();

console.log(fecha);

Explicación del ejemplo

Se obtiene la fecha actual del sistema.

Formularios

Explicación

Los formularios permiten introducir datos. 

function mostrarNombre() {
  let nombre = document.getElementById("nombre").value;
  document.getElementById("resultado").innerHTML = nombre;
}

Explicación del ejemplo

El usuario escribe un nombre y se muestra en pantalla.
